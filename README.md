## Ruby + RSpec + Capybara

#### Install bundler

```gem install bundler```

#### Install gems

```bundle install```

#### Run Rubocop

```rubocop```

#### Run Rubocop with auto refactoring

```rubocop -a```

#### Install Chromedriver on Ubuntu

```https://gist.github.com/ziadoz/3e8ab7e944d02fe872c3454d17af31a5```

```brew cask install chromedriver```
